# Artifact Templates

This directory stores release-engineering templates only. The authoritative
published artifact metadata is the Zenodo record:

- DOI: [10.5281/zenodo.19740954](https://doi.org/10.5281/zenodo.19740954)
- manifest: `raw_results_manifest.json`
- checksums: `SHA256SUMS.txt`

The `*.template.*` files intentionally contain placeholders and should not be
read as the published archival metadata.
